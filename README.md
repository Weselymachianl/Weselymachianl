## 📝 自傳 About Me

**我是陳俞閔**，目前就讀於國立中正大學通訊工程研究所，預計於 2026 年 8 月取得碩士學位。我是一位具備 AI 演算法開發與系統整合實作經驗的應屆畢業生，擅長從數據中分析邏輯，並透過系統化的思維解決技術瓶頸。

大學就讀於國立高雄科技大學通訊工程學系期間，我對演算法設計與系統實作產生濃厚興趣。在大四專題「QR Code 掃描優化系統」中，利用了 Python 開發核心演算法，成功克服了影像干擾與汙損環境下的辨識難題。該專案不僅整合了跨平台網頁展示，更憑藉解決實際痛點的高實用性榮獲畢業專題佳作。

升上碩士後，我的研究聚焦於 **AI 演算法與系統最佳化的應用**。面對流體天線系統（FAS）中複雜的參數優化難題，我透過結合神經網路模型與深度強化學習，讓系統能在「資訊不完整、具備多變數」的環境下，自動學習並尋找最佳的參數配置，以最大化系統效能。在模型開發與測試過程中，透過自行生成的實驗數據進行模型訓練與嚴謹的收斂性分析。這段歷程不僅深化了對機器學習框架（PyTorch）的掌握，更培養了我從數據中梳理邏輯、透過反覆實驗排除異常（Debug）的解題能力。

在技術廣度上，我具備將技術落地並解決實際問題的整合能力。例如在開發多人即時通訊系統時，我運用 Node.js 建構後端，並導入 Docker 容器化技術以確保系統在不同開發環境下都能維持一致的穩定度。秉持著「只要肯學就能掌握」的積極態度，我做事嚴謹、條理分明，善於團隊協作。我期待將這份扎實的學術訓練與重視穩定性的實作精神帶入職場，為企業創造實際的工程價值。

## 🎙️挫折與成就感
【最感挫折的經驗與克服過程】
「我求學過程中最感挫折的經驗，是在碩士研究初期開發 AI 演算法的階段。當時我為了找出最佳參數，一開始採用了純粹的決策型深度強化學習（DRL）。但我很快就遇到了嚴重的效能瓶頸：我發現基礎模型『缺乏記憶性』，它在評估過一種參數組合後，無法有效記住結果，導致訓練過程不斷在原地打轉。

（如何克服）
遇到這個挫折後，我沒有盲目地繼續訓練，而是停下來查閱大量文獻與資料，決定將架構升級為『具備記憶性的深度強化學習』來解決遺忘的問題。然而，挑戰接踵而來，解決了記憶問題後，我發現模型又陷入了『局部最佳解』的陷阱，遲遲無法找到全局最佳解。面對這種連環卡關，我再次深入研究其他領域的解法，最終決定跳脫純 DRL 的框架，大膽導入『預測型的神經網路模型（如 ConvCNP）』來輔助預測與決策，成功引導系統跳出局部盲區，讓模型順利收斂。」

【最有成就感的經驗】
「而這段充滿挫折的過程，剛好也是我求學階段最有成就感的經歷。

對我來說，最有成就感的那一刻，並不是一開始就寫出完美無缺的程式，而是當我經過無數次的『遇到瓶頸 ➡️ 主動尋找文獻 ➡️ 修改底層架構』的努力後，看著最終的測試結果出現了極大幅度的改善，完美實現了我最初設定的研究初衷與系統效能。

這次的經驗讓我深刻體會到，做研究或開發系統，過程一定會遇到理論與實際不符的黑盒子。但我具備了『遇到未知問題時，能獨立尋找資源並轉化為實際解法』的能力，這份突破困難後結出的果實，是我最有成就感的來源。」
